# TikTok

## Front End

0. Install Deps
```
npm install
```

Default User:
```
username: demo
password: 123456
```

1. Setup Mock Api
```
npm install -g json-server
cd test && json-server db.js --port 4000
```

2. Setup SaveFile Endpoint
```
cd test && node saveFile.js
```

3. Start the Frond End
```
npm start
```
